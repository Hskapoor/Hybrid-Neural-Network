## Dataset
http://archive.ics.uci.edu/ml/datasets/statlog+(australian+credit+approval)
the dataset used it statlog from UCI repository

1. Minisom is used to create the map. 
2. Keras
3. Basic libraries such as pandas , numpy , sklearn
